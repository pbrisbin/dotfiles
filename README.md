# Dotfiles

This repo uses [rcm][].

[rcm]: https://github.com/thoughtbot/rcm

## Usage

```
% git clone https://github.com/pbrisbin/dotfiles .pbrisbin-dotfiles
% rcup -d .pbrisbin-dotfiles -x README.md -v
```

See more details with `man 7 rcm`.
